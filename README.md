# mangage-admin

## 主要技术

1. webpack
2. React
3. Ant Design

### 工具

1. IntelliJ IDEA

### 简介
1. 这是一套纯前端的后台商品管理平台的demo
2. 含基础的判断和权限，mock数据在data.js中
3. 由于未使用localStorage/sessionStorage，所以涉及数据变动与后台交互部分未完善

### 目录介绍
1. /src/component |组件|
2. /src/pages |页面js和css目录|
3. /src/router |路由配置|
4. /src/util |工具类函数|
